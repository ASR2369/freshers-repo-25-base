Kharagpur Data Analytics Group

External Description : KDAG is aimed at bringing Data Analytics and Machine Learning enthusiasts together under the umbrella of a single society. The group holds selections each year and conducts workshops and helps assimilate analytics skills in the students. - M

Internal Description : In KDAG (Kharagpur Data Analytics Group), IIT Kharagpur’s official student body for data science and machine learning, you will be engaged deeply with real-world analytics problems and advanced research-oriented work. You get a chance to participate and organise technical workshops, industry-led talks, and it organises India’s largest student-run data science hackathon named as Kharagpur Data Science Hackathon (KDSH) and also organises an internship programme named as ChiSquare. The group’s collaborative environment, combined with rigorous peer mentoring and curated resources, allowing you to explore complex domains such as statistical modelling, deep learning, and algorithmic optimization.

Selection Format : Quiz/Test Round -> Task Round (Single or Multiple tasks) -> Personal Interview

Selection Time : Autumn Sem (Likely September).

Grimoire of Code

External Description : Grimoire of Code, abbreviated as GoC, is IIT Kharagpur's competitive programming club involving a group of passionate competitive programmers. The club conducts global and intra-campus competitive programming contests, and workshops for students on campus, fostering the competitive programming culture within IIT Kharagpur. - M

Internal Description : Grimoire of Code is the Competitive Programming Club of IIT Kharagpur. We are open to all departments and promote the culture of Competitive Programming (CP) in the college. We conduct the Competitive Programming Workshop which helps people of all levels improve their skills (open for all). We also conduct a CP Mentor-Mentee program where you can get a mentor to help you with all your CP related doubts and overall guidance. The flagship event is the CDC series which is a series of 7-8 weekly contests held during the summers to help people prepare for their CDC Online Assessments (i.e. coding tests for internships/placements). We also conduct the lockout contest which is a 1v1 cp contest.
As a member of the club, you get to network with the best competitive programmers on campus, learn from them and get a hand at problem-setting. There is minimal load in the club.

Selection Format : pen-paper based aptitude/math test -> Personal Interview (for freshers). Contest -> PI (for Sophs)

Selection Time : Autumn Sem (Typically October)

Selection Link (when & if applicable) :

Links : Instagram
